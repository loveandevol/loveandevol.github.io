Archlinux 安装

精简的安装步骤,June 10,2021
1,确认启动模式:
ls /sys/firmware/efi/efivars
1.如果存在此文件则是UEFI 2.如果不则是BIOS 我的愚见:在安装过程中两者的分区方案是不一样的,其他没什么影响 2,使用iwctl连上wifi:
iwctl --passphrase 你的密码 station 你的网卡名 connect 你的wifi名
比如: iwctl --passphrase 12345678 station wlp2s0 connect TP_LINK_1234
3,给硬盘分个块(分区):
fdisk 你的磁盘
比如: fdisk /dev/sda, fdisk /dev/nvme0n1
4,给主块装饰一下(格式化):
mkfs.ext4 /dev/主分区
例如:mkfs.ext4 /dev/sda2

如果有交换分区:
mkswap /dev/交换分区
例如:mkswap /dev/sda3
5,连通折腾过的磁盘(挂载):
mount /dev/主分区 /mnt
例如:mount /dev/sda2

如果有交换分区,启用功能:
swapon /dev/交换分区
栗子:swapon /dev/sda3
6,连上了就开始往里面注入一些东西(@#$%!):
pacstrap /mnt base linux linux-firmware grub vim networkmanager
此时一个简易的操作系统就安装完毕了,但如果我要想使用它还需要一些步步 7,生成一个自动连通磁盘的文件fstab(开机自动挂载)
genfstab -U /mnt >> /mnt/etc/fstab
8,走进你的那个磁盘(!@$%!#$)
arch-chroot /mnt
你甚至可以玩一玩:rm -rf / :) :) 9,内核的映射(并不是很懂,有人知道吗)
mkinitcpio -P
10,设置密码(一定要一定要)
passwd
有几次忘了这个,然后只能再进入arch~~.iso(装系统的系统),贼麻烦 :() 当你有了烟,有了心情,却发现没有火,你会发现还是抽不了 :< 11,来个打火机(启动加载器) :) 我选择的是--grub--
grub-install --target=i386-pc /dev/你的磁盘
栗子: 1.BIOS: grub-install --target=i386-pc /dev/sda
grub-install --target=x86_64-efi --efi-directory=esp --bootloader-id=GRUB
栗子: 2.UEFI: grub-install --target=x86_64-efi --efi-directory=esp --bootloader-id=GRUB
12,打火机就绪(配置grub)
grub-mkconfig -o /boot/grub/grub.cfg 完
13,如果不想强行关机,可以先退出chroot
exit
14,在arch~~.iso模式下重启
reboot 真的完
